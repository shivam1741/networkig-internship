# Networking Internship – 3 Months
This repository provides a detailed week-by-week documentation of my 3-month Networking Internship.   
It highlights the key networking concepts I learned, the hands-on labs I practiced in Packet Tracer, the troubleshooting scenarios I solved, and the real-world configurations I implemented during the internship.

### Skills Developed
During this internship, I gained hands-on experience in:

* Basics of computer networks
* LAN, MAN, WAN and common network topologies
* IP addressing, subnetting, MAC addressing, ports & protocols
* OSI & TCP/IP models and core networking concepts (ARP, DNS, DHCP, switching vs routing)
* VLAN creation, access/trunk port configuration, inter-VLAN routing
* Router-on-a-Stick configuration
* Static & Dynamic routing (RIP, OSPF, EIGRP)
* Intra-domain & Inter-domain routing fundamentals
* DHCP setup (both server-based and router-based)
* NAT (Static, Dynamic, PAT)
* ACLs – Standard and Extended
* Basic firewall concepts, rule creation, and VPN fundamentals
* Troubleshooting with ping, traceroute, show commands, interface debugging
* Hands-on use of DXC tools including ticketing systems, monitoring dashboards, remote access tools, and documentation platforms
